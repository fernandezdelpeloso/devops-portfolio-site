--- layout: default title: CI/CD Pipeline Project permalink: /projects/devops-cicd-pipeline.html --- # CI/CD Pipeline with GitHub Actions and Jenkins ## 🔍 Project Overview This project implements a modern **CI/CD pipeline** using **GitHub Actions** and **Jenkins**, automating the process of building, testing, and deploying a containerized application. ## 🧰 Tools & Technologies - **GitHub Actions** - **Jenkins (Docker-based)** - **Docker + Docker Compose** - **Node.js** sample app (or any minimal app stack) - **ngrok** (for exposing Jenkins) ## 📷 Pipeline Architecture ![CI/CD Pipeline Diagram](../assets/cicd-pipeline-diagram-placeholder.png) ## 📁 Repository - GitHub: [devops-cicd-pipeline](https://github.com/fernandezdelpeloso/devops-cicd-pipeline) ## ⚙️ Pipeline Stages - `build`: Compile the app and build the Docker image - `test`: Run automated tests (unit + integration) - `push`: Upload image to Docker Hub - `deploy`: Trigger deployment via Jenkins or GitHub environment ## 🚀 Deployment Instructions 1. Clone the repo: ```bash git clone https://github.com/fernandezdelpeloso/devops-cicd-pipeline.git cd devops-cicd-pipeline ``` 2. Start Jenkins via Docker: ```bash docker-compose up -d ``` 3. Access Jenkins at `http://localhost:8080` 4. Configure GitHub Webhook for push events 5. Commit changes and observe automated pipeline ## ✅ GitHub Actions Example ```yaml name: CI/CD Pipeline on: push: branches: - main jobs: build-test-deploy: runs-on: ubuntu-latest steps: - uses: actions/checkout@v3 - name: Run tests run: npm test - name: Build Docker image run: docker build -t user/app . - name: Deploy (trigger Jenkins) run: curl -X POST http://jenkins/job/deploy/build ``` ## 🧠 Lessons Learned - Parallelizing GitHub Actions jobs - Managing secrets with GitHub Actions - Jenkins integration with Docker and GitHub - Docker image reuse and tagging strategies ## 📈 Next Steps - Add Slack notifications for failures - Add staging/prod environment workflows - Configure rollback and test coverage reporting --- **Back to [Home](../index.html)**
